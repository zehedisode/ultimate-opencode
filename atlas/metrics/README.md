# ATLAS Metrics

Bu dizin ATLAS metrik modülünü içerir.

## Dosyalar

| Dosya | Açıklama |
|---|---|
| `costs.md` | Token tüketimi ve USD bazında maliyet takibi |
| `heatmap.md` | Dosya değişim sıklığı ve risk haritası |
| `performance.md` | İşlem süreleri ve performans metrikleri (YENİ) |

## Kullanım

Bu dosyalar ATLAS tarafından otomatik okunur ve güncellenir.
