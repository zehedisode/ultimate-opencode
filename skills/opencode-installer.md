# Opencode Installer

Tek adımda opencode CLI ve plugin kurulumu yapan araç.

## Ne İşe Yarar

- opencode'u sıfırdan kur
- Plugin'leri otomatik yükle
- MCP server'ları yapılandır
- Sistem gereksinimlerini kontrol et

## Kullanım

```bash
opencode-installer                    # Tümünü kur
opencode-installer --minimal          # Sadece opencode
opencode-installer --plugins gitlab   # Belirli plugin'ler
```

## Özellikler

- Platform algılama (Linux/macOS/WSL)
- Bağımlılık kontrolü
- İnteraktif kurulum sihirbazı
- Rollback desteği
