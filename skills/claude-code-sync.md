# Claude Code Sync

Claude Code oturumları arasında context ve state senkronizasyonu.

## Ne İşe Yarar

- Çoklu terminal senkronizasyonu
- Oturum kurtarma
- Context paylaşımı
- İşbirliği desteği

## Kullanım

```bash
claude-code-sync push     # Mevcut context'i gönder
claude-code-sync pull     # Başka oturumdan al
claude-code-sync status   # Senkronizasyon durumu
```

## Özellikler

- Gerçek zamanlı senkronizasyon
- Çakışma çözümü
- Context versiyonlama
- Güvenli SSL iletişim
