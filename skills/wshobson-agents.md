# WS Hobson Agents

Multi-harness agentic plugin marketplace. Claude Code, Codex, Cursor, OpenCode, Copilot için.

## Ne İşe Yarar

- Plugin marketplace
- Cross-harness uyumlu
- Tek komut kurulum
- Topluluk testli

## Kullanım

```bash
agents search "react"       # Plugin ara
agents install <id>         # Plugin kur
agents list                 # Yüklü plugin'ler
agents update               # Güncelle
```
