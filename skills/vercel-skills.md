# Vercel Skills

`npx skills` ile skill yönetimi ve GitHub'dan skill yükleme aracı.

## Ne İşe Yarar

- GitHub'dan skill bul ve yükle
- Skill kataloğunu keşfet
- Skill bağımlılıklarını yönet

## Kullanım

```bash
npx skills                 # Kataloğu göster
npx skills install <ad>    # Skill yükle
npx skills search <terim>  # Skill ara
npx skills update          # Skill'leri güncelle
```
