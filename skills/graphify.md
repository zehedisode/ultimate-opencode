# Graphify

Herhangi bir klasörü knowledge graph'a dönüştüren araç.

## Ne İşe Yarar

- Kod tabanını grafik olarak görselleştir
- Bağımlılık ilişkilerini keşfet
- Modül yapısını anla
- Otomatik dökümantasyon

## Kullanım

```bash
graphify init              # Mevcut dizinde grafik oluştur
graphify explore <sembol>  # Sembolü grafikte bul
graphify viz               # Görselleştir
graphify stats             # İstatistik göster
```

## Özellikler

- Dil-agnostik (Python, JS, Go, Rust, Java)
- Mermaid.js çıktı
- Otomatik güncelleme
- CI/CD entegrasyonu
