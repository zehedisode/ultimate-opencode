# Opencode Cost Optimizer

opencode token tüketimini analiz eden ve optimize eden araç.

## Ne İşe Yarar

- Token tüketim raporu
- En pahalı işlemleri tespit
- Context compression önerileri
- Model değiştirme önerileri

## Kullanım

```bash
oco analyze                # Token analizi
oco report --weekly        # Haftalık rapor
oco optimize               # Optimizasyon önerileri
oco suggest-model          # En uygun modeli öner
```

## Özellikler

- Gerçek zamanlı token takibi
- Maliyet tahmini
- Model bazlı karşılaştırma
- İyileştirme puanı
