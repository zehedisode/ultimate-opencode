# Gemini CLI

Google Gemini model'lerini terminalden kullanmak için CLI aracı.

## Ne İşe Yarar

- Gemini model'lerine terminal erişimi
- Multi-modal (text, image, code)
- Streaming output
- Code execution

## Kullanım

```bash
gemini "merhaba dünya"
gemini --model gemini-2.5-pro
gemini --image photo.jpg "bunu açıkla"
```

## Özellikler

- 1M+ context window
- File upload (image, PDF, code)
- Function calling
- MCP entegrasyonu
