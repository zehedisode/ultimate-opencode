# Dify

Production-ready agentic workflow platform. Visual builder, RAG, MCP native.

## Ne İşe Yarar

- Görsel AI workflow oluşturma
- RAG pipeline yönetimi
- Multi-agent orchestration
- MCP native entegrasyon

## Kullanım

```bash
dify start          # Dify'ı başlat
dify workflow new   # Yeni workflow oluştur
dify deploy         # Dağıt
```

## Entegrasyon

MCP üzerinden opencode ile çalışır.
