# OpenCLI — Website → CLI

Herhangi bir website'yi CLI komutuna çeviren araç.

## Ne İşe Yarar

- Website'leri terminal'den kullan
- curl/wget ile etkileşim
- Sayfa içeriğini parse et
- API endpoint'lerini keşfet

## Kullanım

```bash
opencli <url> <komut>
opencli https://opencode.ai "show docs"
```

## Özellikler

- Sayfa içeriğini CLI argümanına çevirir
- Markdown çıktı formatı
- Hızlı prototipleme ve test
