# Opencode MCP Hub

opencode için MCP server marketplace ve merkezi yönetici.

## Ne İşe Yarar

- MCP server keşfet ve kur
- Tek komutla MCP yönetimi
- Version kontrolü
- Bağımlılık çözümleme

## Kullanım

```bash
mcp-hub search "database"     # MCP ara
mcp-hub install postgres      # MCP kur
mcp-hub list                  # Yüklü MCP'ler
mcp-hub update all            # Tümünü güncelle
```

## Özellikler

- 100+ MCP server katalog
- Tek satır kurulum
- Otomatik yapılandırma
- Sağlık kontrolü
