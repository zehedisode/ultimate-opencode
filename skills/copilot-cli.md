# GitHub Copilot CLI

Terminal'de AI destekli komut önerileri ve açıklamaları.

## Ne İşe Yarar

- Komut açıklaması: `whatis "dosyaları bul ve sil"`
- Komut önerisi: `suggest "docker container'ları listele"`
- Alias yönetimi
- CLI entegrasyonu

## Kullanım

```bash
gh copilot explain "docker ps -a"
gh copilot suggest "find all large files"
```

## Özellikler

- Doğal dil → komut dönüşümü
- Çoklu shell desteği
- Pipe-aware öneriler
- Git-aware
