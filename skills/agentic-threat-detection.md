# Agentic Threat Detection

AI agent'larda güvenlik tehditlerini tespit eden framework.

## Ne İşe Yarar

- Prompt injection tespiti
- Data exfiltration önleme
- Tool abuse detection
- Session hijacking koruması

## Kullanım

```bash
atd scan prompt "..."       # Prompt'u tara
atd monitor session         # Session'ı izle
atd audit logs              # Logları denetle
```

## Tehdit Kategorileri

- Prompt injection
- Indirect prompt injection
- Data exfiltration
- Tool abuse
- Session hijacking
