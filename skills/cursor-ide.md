# Cursor IDE

AI-first code editor. VS Code tabanlı, built-in AI asistan.

## Ne İşe Yarar

- AI code editing
- Multi-model (GPT-4, Claude, Gemini)
- Agent mode
- Codebase indexing

## Kullanım

```bash
cursor .                 # Projeyi aç
cursor --no-sandbox      # Güvenli mod
cursor --install-extension <id>
```

## Özellikler

- Cmd+K ile AI düzenleme
- Chat + Composer
- Codebase RAG
- Custom rules
- VS Code extension uyumlu
