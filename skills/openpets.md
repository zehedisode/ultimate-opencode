# OpenPets — Terminal Pet

Terminal'de sanal evcil hayvan besleme aracı.

## Ne İşe Yarar

- Kod yazarken moral desteği
- Eğlenceli terminal animasyonları
- Pet durum göstergeleri (mutluluk, açlık, enerji)
- Çalışma süresi takibi

## Kullanım

```bash
openpets               # Pet'i başlat
openpets --feed        # Besle
openpets --status      # Durum göster
```

## Özellikler

- ASCII art animasyonlar
- 10+ farklı pet türü
- Çalışma istatistikleri
- Minimal CPU kullanımı
