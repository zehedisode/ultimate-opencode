# Aider

AI pair programming terminal aracı. Multi-model, git-aware, code editing.

## Ne İşe Yarar

- Doğal dil ile kod yazma
- Multi-model (GPT-4, Claude, DeepSeek)
- Git-aware otomatik commit
- Map of repo ile kod anlama

## Kullanım

```bash
aider --model claude-3-opus   # Claude ile başlat
aider src/main.py             # Belirli dosyada çalış
aider --dark-mode             # Karanlık tema
```

## Özellikler

- 10+ model desteği
- Otomatik git commit
- LINT entegrasyonu
- Map of repo (repo haritası)
