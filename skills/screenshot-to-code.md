# Screenshot to Code

Ekran görüntüsünden veya tasarımdan kod üreten AI aracı.

## Ne İşe Yarar

- Tasarım → HTML/CSS/React/Vue dönüşümü
- Prototipleme hızlandırma
- UI keşfi ve reverse engineering
- Çoklu framework desteği

## Kullanım

```bash
screenshot-to-code input.png   # Görselden kod üret
screenshot-to-code --react     # React çıktı
screenshot-to-code --tailwind  # Tailwind CSS çıktı
```

## Desteklenen Çıktılar

- HTML + CSS
- React + Tailwind
- Vue + SCSS
- Bootstrap
- Material UI
