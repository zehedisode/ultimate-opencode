# Firecrawl MCP

Web scraping ve arama için MCP sunucusu.

## Ne İşe Yarar

- Web sayfalarını scrape et
- Site haritası çıkar
- Markdown dönüşümü
- LLM-ready içerik

## Kullanım

MCP üzerinden otomatik çalışır.

```bash
npx firecrawl-mcp
```

## Özellikler

- JavaScript rendering
- Rate limiting
- Çoklu sayfa tarama
- Custom selector desteği
