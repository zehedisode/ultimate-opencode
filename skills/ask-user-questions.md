# Ask User Questions

Kullanıcıya net, odaklı sorular sormak için kullanılan araç.

## Ne İşe Yarar

- Belirsiz gereksinimleri netleştirir
- Kullanıcı tercihlerini öğrenir
- Karar noktalarında kullanıcıyı bilgilendirir
- Proje yönünü kullanıcı onayıyla belirler

## Kullanım

```bash
/question "Hangi framework'ü kullanmak istersiniz?"
```

## Özellikler

- Çoktan seçmeli soru desteği
- Açık uçlu soru desteği
- Varsayılan öneri gösterme
- İlerleme takibi
