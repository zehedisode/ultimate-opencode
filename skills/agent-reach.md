# Agent Reach

AI agent'lara internet görüşü kazandıran araç. Twitter, Reddit, YouTube, GitHub — API'siz.

## Ne İşe Yarar

- Sosyal medya verisi toplama
- GitHub trending takibi
- YouTube içerik analizi
- Reddit gönderi okuma

## Kullanım

```bash
agent-reach twitter "AI news"         # Twitter'da ara
agent-reach github trending           # GitHub trending
agent-reach reddit r/MachineLearning  # Reddit oku
```

## Özellikler

- API key gerektirmez
- Rate limiting yok
- MCP uyumlu
- JSON çıktı
