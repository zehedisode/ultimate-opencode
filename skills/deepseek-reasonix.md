# DeepSeek Reasonix

DeepSeek-native AI coding agent. Terminal TUI, prefix-cache stabil.

## Ne İşe Yarar

- DeepSeek modelleri ile kod geliştirme
- Terminal TUI arayüzü
- Prefix-cache ile hızlı inference
- Codex/OpenCode uyumlu

## Kullanım

```bash
reasonix serve          # Sunucuyu başlat
reasonix code "..."     # Kod oluştur
reasonix chat           # Sohbet modu
```
