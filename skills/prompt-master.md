# Prompt Master

AI asistanlar için kusursuz prompt yazma aracı ve kılavuzu.

## Ne İşe Yarar

- Etkili prompt şablonları
- Chain-of-thought yapılandırması
- Rol atama ve context yönetimi
- Çıktı formatı kontrolü

## Teknikler

1. **Rol Belirleme**: "Sen bir Python uzmanısın..."
2. **Bağlam Sağlama**: Çalışma dizini, kod tabanı, hedef
3. **Çıktı Formatı**: JSON, Markdown, kod bloğu
4. **Kısıtlar**: Token limiti, dil, stil kuralları

## Örnek

```markdown
Sen bir kod inceleme uzmanısın.
Şu dosyayı güvenlik, performans ve okunabilirlik açısından değerlendir.
Çıktıyı Markdown tablo formatında ver.
```
