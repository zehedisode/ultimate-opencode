# Agentify Desktop

Browser session kontrol ve masaüstü otomasyon aracı.

## Ne İşe Yarar

- Browser tab/session yönetimi
- DOM element sorgulama
- Sayfa etkileşimleri (tıkla, yaz, bekle)
- Ekran görüntüsü alma
- Session state izleme

## Kullanım

```bash
agentify-desktop --browser chrome "find .btn-submit"
agentify-desktop --screenshot
agentify-desktop --session list
```

## Özellikler

- Çoklu browser desteği (Chrome, Firefox, Edge)
- Headless mod
- Session persistence
- API üzerinden kontrol
