# Lazygit — Terminal UI Git Client

Terminal için görsel git arayüzü.

## Ne İşe Yarar

- Git işlemleri için TUI
- Görsel diff/staging
- Branch yönetimi
- Cherry-pick, rebase, merge

## Kullanım

```bash
lazygit              # TUI'yi aç
lazygit --git-dir    # Farklı repo
lazygit --help       # Yardım
```

## Özellikler

- Fare desteği
- Custom keybinding
- Çoklu panel görünümü
- AI agent dostu CLI
