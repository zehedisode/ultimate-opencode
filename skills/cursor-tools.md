# Cursor Tools

Cursor IDE ile opencode arasında köprü kuran araç seti.

## Ne İşe Yarar

- Cursor kurallarını opencode'a aktar
- `.cursorrules` → opencode AGENTS.md dönüşümü
- İki yönlü context senkronizasyonu
- Ortak komut paleti entegrasyonu

## Kullanım

```bash
cursor-tools sync          # Kuralları senkronize et
cursor-tools export        # Cursor'dan opencode'a aktar
cursor-tools watch         # Değişiklikleri izle
```

## Özellikler

- Otomatik kural dönüşümü
- Çift yönlü senkronizasyon
- CLI + TUI arayüzü
- Minimal konfigürasyon
