# Open Interpreter

Terminal için doğal dil bilgisayar arayüzü. Code, shell, web, dosya işlemleri.

## Ne İşe Yarar

- Doğal dil ile bilgisayar kontrolü
- Kod yazma ve çalıştırma
- Dosya işlemleri
- Web scraping
- API çağrıları

## Kullanım

```bash
interpreter                    # Sohbet başlat
interpreter "plot this data"   # Veriyi görselleştir
interpreter --model gpt-5.4    # Model seç
```

## Özellikler

- 20+ dil desteği
- Safe mode
- Docker entegrasyonu
- OS kontrol
